# ElastiCache

## intro

- used for distributed in-memory cache environments in AWS
- runs either Redis or Memcached

## comparison to DynamoDB

| Elasticache          | DynamoDB            |
| -------------------- | ------------------- |
| micro second latency | millisecond latency |
| in memory            | on disk             |
